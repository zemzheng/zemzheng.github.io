推荐插件

  • vim-pathogen 插件管理工具，配置完成后把插件下载到定义好的目录即可
  • emmet-vim HTML 快捷生成工具，顺便还可以定义自己的快捷语法，曾经的 Zen-Coding
  • tabular 对齐工具，洁癖患者福音
  • jshint.vim js 代码检查工具
  • vim-airline 分割窗口后提供底部小标识，方便查看
  • vim-coffee-script coffeescript 的 vim 语法高亮
  • vim-less less css 的语法高亮
